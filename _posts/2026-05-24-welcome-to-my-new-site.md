# Hello World!

This is my first post using the **Creative Theme** for Jekyll. 

Because we added a `page.html` layout, I can now write content in **Markdown** and it will be rendered beautifully inside the site's design.

## Why use Markdown?

1. **Simplicity**: Focus on writing, not HTML tags.
2. **Speed**: It's much faster to draft a CV or a blog post.
3. **Consistency**: The layout handles the styling, so every page looks the same.
